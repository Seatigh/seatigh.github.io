Apps

Applicaciones

I have made a desktop application for Windows to take control of the invoices for freelancers.

You can add, search, modify and remove invoices. Also, you can have control of the different clients by adding them (future features: edit and remove clients).

Additionally, there are two tabs to check the entries and the outlays by client or by period.

This application works with an internal data base created at the first time you use it. You can create new data bases if you need to control the invoices of another personal activity.

He hecho una aplicación de escritorio para Windows para poder llevar el control de las facturas de un autónomo o empresa.

Puedes añadir, buscar, editar y borrar facturas, añadir a tus clientes para llevar un seguimiento completo de las facturas que emites para cada uno de ellos, así como poder comprobar los ingresos y gastos por cliente y por trimestre.

Esta aplicación funciona con una base de datos interna que se crea al iniciar la aplicación por primera vez. Posteriormente se pueden crear nuevas bases de datos para llevar el control de otras actividades.
